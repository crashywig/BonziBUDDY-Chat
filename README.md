# BonziBUDDY Chat

## Dependencies
- Node.js and npm
- Git

## Setup
In a terminal/command prompt, navigate to where you'd like BonziBUDDY Chat to be placed and run the following:
```
git clone https://github.com/crashywig/BonziBUDDY-Chat
cd BonziBUDDY-Chat
```

### Client
This part is unneeded. A pre-built version is available.

### Server
```
cd server
npm install
node index.js
```
After this, BonziBUDDY Chat will be accessible on port 3000. (http://localhost:3000/)
